# Переменные в Python

Переменные используются для хранения данных в памяти программы. В Python переменная создается в момент присваивания ей значения.

## Объявление переменных

```python
имя_переменной = значение
```

### Пример

```python
x = 10
y = "Привет"
z = 3.14

print(x, y, z)

# Вывод: 10 Привет 3.14
```

## Правила именования переменных

- Имя переменной может содержать буквы, цифры и символ `_`, но не может начинаться с цифры.
- Чувствительно к регистру (`name` и `Name` — разные переменные).
- Не должно совпадать с ключевыми словами Python (например, `if`, `for`, `while`).

## Типы данных

В Python переменные не требуют явного указания типа, но у каждой переменной есть тип.

### Основные типы данных:

- `int` — целые числа (`x = 10`)
- `float` — числа с плавающей запятой (`y = 3.14`)
- `str` — строки (`z = "Hello"`)
- `bool` — логические (`True` или `False`)
- `list` — списки (`arr = [1, 2, 3]`)
- `tuple` — кортежи (`t = (1, 2, 3)`)
- `dict` — словари (`d = {"ключ": "значение"}`)
- `set` — множества (`s = {1, 2, 3}`)

## Проверка типа переменной

```python
x = 42
print(type(x))  # Вывод: <class 'int'>
```

## Динамическая типизация

Python позволяет изменять тип переменной в процессе выполнения:

```python
x = 10 x = "Теперь я строка!" print(x)
# Вывод: Теперь я строка!
```

## Множественное присваивание

Можно присваивать несколько значений сразу:

```python
a, b, c = 1, 2, 3 print(a, b, c)
# Вывод: 1 2 3
```

## Заключение

Переменные — основа любой программы. Python позволяет работать с ними гибко благодаря динамической типизации и множественным встроенным типам данных.
