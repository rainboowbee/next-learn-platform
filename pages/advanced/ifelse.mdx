# Условные конструкции в Python

В Python условные конструкции (`if`, `elif`, `else`) позволяют выполнять код в зависимости от условий.

## Синтаксис

```python filename="demo.py" copy
if условие:
    # Код, который выполнится, если условие истинно
elif другое_условие:
    # Код, если первое условие ложно, но это истинно
else:
    # Код, если все предыдущие условия ложны
```

## Примеры использования

### Простое условие

```python filename="demo.py" copy
x = 10 if x > 5: print("x больше 5")
# Вывод: x больше 5
```

### if-elif-else

```python filename="demo.py" copy
x = 0
if x > 0:
    print("Положительное число")
elif x < 0:
    print("Отрицательное число")
else:
    print("Ноль")
# Вывод: Ноль
```

# Логические операции: and, or, not

Логические операторы позволяют объединять условия:

- `and` – истина, если оба условия истинны.
- `or` – истина, если хотя бы одно условие истинно.
- `not` – инвертирует логическое значение.

## Примеры использования

### Оператор and

```python
  x = 10 y = 5 if x > 0 and y > 0: print("Оба числа положительные") # Вывод: Оба
  числа положительные
```

### Оператор or

```python
  x = -10 y = 5 if x > 0 or y > 0: print("Хотя бы одно число положительное") #
  Вывод: Хотя бы одно число положительное
```

### Оператор not

```python
x = False if not x: print("x равно False")
# Вывод: x равно False
```

## Заключение

Условные конструкции и логические операторы — важные инструменты в программировании на Python. Они позволяют управлять выполнением кода в зависимости от условий.
